# How to clone project


- Download by ZIP

```
[ZIP](https://github.com/deeppaz/serenes/archive/master.zip)
```

- or clone this repo

```
git clone https://github.com/deeppaz/serenes.git
```
```
cd serenes
```

```
npm install
```

```
npm start
```
